What it is

Debian —

but beautiful.

BorealOS starts where Debian leaves off. We pre-configure the looks of every DE/WM so you can get to work right away. The base is solid and familiar, but the init system is swapped out for OpenRC — a lightweight init system and service manager for Unix-like operating systems. It starts services based on dependencies, supports parallel booting, and continuously supervises services, restarting them if needed. Unlike systemd, it focuses only on initialization and service management, making it fast, simple, and resource-efficient.

The project targets intermediate Linux users who want a clean, well-documented system with pre-configured desktops and a community-driven development model. No hand-holding, no unnecessary abstraction.

The shell defaults to sh, with fish available as options at setup. Desktops and window managers come pre-configured. Documentation is treated as a first-class deliverable, with a focus on man pages.

What sets it apart

Key Features

OpenRC

A lightweight init system that starts and supervises services using dependency-based, parallel startup, providing a simple and efficient alternative to systemd.

Pre-configured Desktops

Choose from KDE Plasma, XFCE, and Niri iso installations. Each comes configured out of the box — no post-install configuration required.

Debian Foundation

Built on one of the most stable and widely-supported Linux bases. The package ecosystem, tooling, and community come along for the ride.

Safety Warnings

Risky operations are flagged before execution. Not babysitting — just giving you the chance to catch mistakes before they matter.

Documentation First

Man pages are a priority, not an afterthought. Coverage for installation, init system usage, desktop customization, and troubleshooting.

Community Driven

Open development with a transparent roadmap. Contributions follow a structured process with pull request templates and clear ownership.
